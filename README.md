# Xprowess - Static HTML/CSS/JS Website

Built with pure HTML, CSS, and JavaScript. No build step needed.

## Local Development
Open `index.html` directly in your browser.

## Deployment (Netlify)
```bash
git add .
git commit -m "feat: complete static HTML/CSS/JS website with 10 pages"
git push
```
Netlify auto-deploys from the root directory.
